Project: naive-ticket-machine
Authors: David Barnes and Michael Kolling, rev. Patrick Girard

This project is part of the material for the OOD-OOP course from the GPhy master programme

It is inspired by the naive-ticket-machine project, chapter of the book:

   Objects First with Java - A Practical Introduction using BlueJ
   Sixth edition
   David J. Barnes and Michael Kolling
   Pearson Education, 2017

Purpose of project: To illustrate the basics of fields, constructors, and methods.
How to start this project: Create one or more TicketMachine objects.
